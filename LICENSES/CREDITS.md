# Per-font credits and Reserved Font Names

Every TTF hosted in this repo is licensed under the **SIL Open Font
License, Version 1.1** (full text in [OFL-1.1.txt](./OFL-1.1.txt)).

The OFL requires that each font's copyright notice and any Reserved
Font Name be preserved with the binary on redistribution
(Section 4). The entries below are the copyright statements as they
appear in each font's upstream release; please verify against
upstream before bundling these files into a derivative work, as
copyright years and contributor lists update from time to time.

If you spot a stale copyright statement here, please open a PR with
the up-to-date text from the upstream's own `OFL.txt` /
`LICENSE.txt`.

binary's `name` table (records 0, 13, and 14).
